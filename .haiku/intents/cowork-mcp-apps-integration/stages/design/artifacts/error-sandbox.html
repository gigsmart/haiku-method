Blocked: clipboard-write
The review panel needs clipboard-write access to let you copy the review link. Your host has not granted this permission to the iframe sandbox. You can still select and copy text manually.
Error code: SANDBOX_CLIPBOARD_WRITE_DENIED

What happened

MCP Apps iframes run inside a sandboxed context. The host controls which browser APIs the iframe can access via the sandbox attribute on the <iframe> element. Clipboard write access is not included by default.

Sandbox attribute received

allow-scripts allow-same-origin allow-forms

How to recover

Select the text in any input field below and copy it with your keyboard (Ctrl+C / ⌘C). Your host administrator can grant allow-clipboard-write to remove this restriction.
