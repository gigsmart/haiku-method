# Report

## Criteria Guidance

Good criteria examples:
- "Executive dashboard shows project health with red/amber/green indicators backed by quantitative thresholds, not subjective judgment"
- "Stakeholder report tailors detail level to audience — executives get 1-page summary, team leads get work-package detail"
- "Forecast section projects completion based on current velocity, not the original plan"

Bad criteria examples:
- "Report is generated"
- "Dashboard is updated"
- "Stakeholders are informed"

## Completion Signal

Project dashboard exists with health indicators, progress visualization, and forecast. Reporter has confirmed all metrics are current and accurately represent project state. Communicator has tailored reports to each stakeholder audience and flagged items requiring stakeholder decisions or escalation.
