# ReactConf Talk: Server Components in Production

25-minute ReactConf talk on shipping React Server Components in production. Audience has read the docs but hasn't built anything real yet — the talk bridges the theory/practice gap with one anchor demo and a narrative arc around the tradeoffs we hit.

## Goals

- Narrative arc: what we expected → what broke → what we learned
- One anchor live demo (streaming data + RSC boundaries) that runs in under 3 minutes
- Code walkthroughs in slides for the non-demo content
- Post-talk blog + repo drop within 48 hours
