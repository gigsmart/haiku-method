# Game Development

Lifecycle for game development. Differs from application development in three
ways: concept absorbs discovery (game pitches and market fit are inseparable),
prototype is a gated validation stage (if the core loop isn't fun, production
is wasted), and polish is its own dedicated stage because game feel and
tuning need iteration time that app work does not.
