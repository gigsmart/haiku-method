# Mobile App v4.0 Release Certification

Pre-release QA certification for the mobile app v4.0 on iOS and Android. Scope is smoke + impacted-areas regression plus the full accessibility suite. Localization runs on its own track and is out of scope here.

## Goals

- Test plan covering smoke suite + impacted areas from the v4.0 changelog
- Full a11y audit against WCAG 2.2 AA (screen reader, contrast, focus order)
- Bug triage and severity gating — zero critical/high at sign-off
- Signed certification memo for the release manager
