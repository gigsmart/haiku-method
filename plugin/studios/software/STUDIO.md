# Application Development

Lifecycle for user-facing applications — web, mobile, desktop, and services.
Covers product discovery, UX design, implementation, operations, and security
review. For libraries, SDKs, and CLI tools, use `libdev`. For games, use
`gamedev`. For hardware products, use `hwdev`.

Supports both single-stage (all disciplines merged) and multi-stage
(sequential discipline progression) execution modes.
